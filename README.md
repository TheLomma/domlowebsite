# Max Musterzauberer – Website

Offizielle Website von Max Musterzauberer, Magician & Mentalist.

Gebaut als statische HTML-Website. Kein Framework, kein Build-Schritt notwendig.

## Lokal ansehen

Einfach `index.html` im Browser öffnen – fertig.

## Deployment via Vercel

1. Repository auf GitHub erstellen
2. Bei [vercel.com](https://vercel.com) anmelden
3. „Add New Project" → GitHub-Repo auswählen
4. Framework: **Other** (kein Framework)
5. Root Directory: `/` (Standard)
6. Deploy klicken – fertig!

## Eigene Domain verbinden

In Vercel unter „Settings → Domains" die eigene Domain eintragen
und die angezeigten DNS-Einträge beim Domain-Anbieter setzen.
